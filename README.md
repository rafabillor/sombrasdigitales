# Vibe Coding - Aplicación Web Sencilla

Estructura base para iniciar rápidamente una aplicación web estática con HTML, CSS y JavaScript.

## Estructura de carpetas

```
Vibe Coding/
├── index.html        # Punto de entrada principal
├── css/
│   └── styles.css    # Estilos globales de la interfaz
├── js/
│   └── main.js       # Interactividad básica del sitio
└── assets/
    └── images/       # Carpeta para imágenes y recursos multimedia
```

## Cómo usar

1. Abre `index.html` en tu navegador preferido.
2. Personaliza los estilos en `css/styles.css` según tu identidad visual.
3. Añade funcionalidad adicional en `js/main.js`.

¡Listo! Ya tienes una base organizada para seguir desarrollando tu aplicación.
